MetaPhylo.treelength — Method
treelength(tree::Tree, [idx::Integer])

Return maximum distance from the root to the leaves in the tree. If the index is specified, this returns maximum distance from the specified idx to its leaves in the tree. The Tree branch types must have the Length trait. See also treeheight.
